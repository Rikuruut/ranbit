# AGENTS.md

## 最重要：Expo SDK 54を前提にする

このプロジェクトは Expo SDK 54 を使用しています。

ExpoはSDKバージョンごとに仕様や対応パッケージが変わるため、コードを書く前に必ず以下のバージョン固定ドキュメントを前提にしてください。

https://docs.expo.dev/versions/v54.0.0/

古いExpo SDKの知識や、最新版SDKの情報だけで実装しないでください。

特に以下はSDK 54に対応するバージョンを優先してください。

- expo
- react
- react-dom
- react-native
- expo-router
- react-native-reanimated
- react-native-worklets
